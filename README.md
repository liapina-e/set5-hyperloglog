# Проект: Анализ точности алгоритма HyperLogLog

## Описание проекта

Данный проект реализует исследование точности вероятностного алгоритма **HyperLogLog** для оценки количества уникальных элементов в потоке данных. Проект состоит из двух частей:
1. **C++ программа** для генерации данных и вычисления оценок
2. **Python скрипт** для визуализации результатов

## Структура файлов

### 1. C++ файлы
- **main.cpp** - основная программа, содержащая:
  - Класс `RandomStreamGen` для генерации потоков данных
  - Класс `HashFuncGen` для генерации хеш-функций
  - Класс `HyperLogLog` - реализация алгоритма оценки уникальных элементов
  - Функции для анализа точности и сохранения данных

### 2. Python файлы
- **script.py** - скрипт для построения графиков

### 3. Файлы данных (генерируются программой)
- **stream_small_plot_data.csv** - данные для маленького потока
- **stream_medium_plot_data.csv** - данные для среднего потока  
- **stream_large_plot_data.csv** - данные для большого потока
- **aggregated_stats.csv** - агрегированные статистики по всем потокам

### 4. Файлы графиков (генерируются скриптом)
- **hyperloglog_comparison.png** - График №1: Сравнение точных значений и оценок
- **hyperloglog_statistics.png** - График №2: Статистики оценки с областью неопределенности
